Tabs


Example 1 - Basic Tabs


Each tab has an <a> with a link to a page. On that page, set that tab as "active" using class="active"

  • Home
  • Profile
  • Messages

// markup
<ul class="nav nav-tabs">
  <li class="active"><a href="#">Home</a></li>
  <li><a href="#">Profile</a></li>
  <li><a href="#">Messages</a></li>
</ul>

Example 2 - Dynamic Tabs


Transition through panes of local content. <a href=""> or data-target points to the element id of the content container. Each "tab" has to be activated via javascript.

  • Home
  • Profile
  • Messages
  • Settings

Home

homey content here

Profile

profiley stuff here

Messages

messages here

Settings

config me!

// script
$(function() {
  $('#myTab a').click(function (e) {
    e.preventDefault();
    $(this).tab('show');
  })
});

// markup
<ul class="nav nav-tabs" id="myTab">
  <li class="active"><a href="#home">Home</a></li>
  <li><a href="#profile">Profile</a></li>
  <li><a href="#messages">Messages</a></li>
  <li><a href="#settings">Settings</a></li>
</ul>
 
<div class="tab-content">
  <div class="tab-pane active" id="home"><h4>Home</h4>homey content here</div>
  <div class="tab-pane" id="profile"><h4>Profile</h4>profiley stuff here</div>
  <div class="tab-pane" id="messages"><h4>Messages</h4>messages here</div>
  <div class="tab-pane" id="settings"><h4>Settings</h4>config me!</div>
</div>
