Navbar


Example 1 - Simple Nav Bar


Navbars are static (not fixed to the top) and include support for a project name and basic navigation.

Have a Brand link to your home page using class="brand" href="#"

Identify selected link with class="active"

Create link dividers with <li class="divider-vertical"></li>

Title
  • Home
  • Profile
  • Settings

<div class="navbar">
  <div class="navbar-inner">
    <a class="brand" href="#">Title</a>
    <ul class="nav">
      <li class="active"><a href="#">Home</a></li>
      <li><a href="forms.html">Profile</a></li>
      <li class="divider-vertical"></li>
      <li><a href="index.html">Settings</a></li>
    </ul>
  </div>
</div>

Example 2 - Nav Bar Forms


To properly style and position a form within the navbar, add the appropriate classes as shown below.


<div class="navbar">
  <div class="navbar-inner">  
    <form class="navbar-form pull-right">
      <input type="text" class="span2">
      <button type="submit" class="btn">Submit</button>
    </form>
  </div>
</div>
