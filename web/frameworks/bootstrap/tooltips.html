Bootstrap Framework Demo

Tooltips

Tooltips are opt-in and must be intialized

Example 1 - Tooltip via HTML Attribues


Tooltip on top  Tooltip on right 


//
// init on doc load
//
$(".ttExample").each(function(){
  $(this).tooltip();
});

//
// markup
//
<a class="ttExample" href="#" data-toggle="tooltip" data-placement="top" title="" data-original-title="Tooltip on top">
Tooltip on top</a> 
<a class="ttExample" href="#" data-toggle="tooltip" data-placement="right" title="" data-original-title="Tooltip on top">
Tooltip on right</a>

Example 2 - Tooltip via Javascript


Tooltip me 

//
// init on doc load
//
$("#ttJsEx").tooltip({
  placement:'bottom',
  title:'dynamic tooltip is how I roll'
});

//
// markup
//
<span id="ttJsEx" >Tooltip me</span>

Example 3 - In a table

You do not want to tooltip a <td> tag. It breaks the layout of the table when the tooltip displays. You want to wrap the item in a span etc.

Hover over the B column rows
Foo Bar
A B
C D 

//
// init on doc load
//
$(".ttSpan").each(function(){
  $(this).tooltip({
    placement:'right',
    title:'tooltip in a table'
  });
});


//
// markup
//
<table class="table table-condensed" style="width: 50%; margin: auto;">
  <thead>
    <tr>
      <th>Foo</th>
      <th>Bar</th>
    </tr>
  </thead>
  <tbody>
  <tr>
    <td>A</td>
    <td><span class="ttSpan">B</span></td>
  </tr>
 <tr>
    <td>C</td>
    <td><span class="ttSpan">D</span></td>
  </tr>  
</tbody>
</table>
