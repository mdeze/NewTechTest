Popover

Popovers are opt-in and must be intialized

Example 1 - popover via HTML Attribues



//
// init on doc load
//
$("#popEx1").popover();

//
// markup
//
<button class="btn btn-small btn-primary" id="popEx1" title='Popver title' data-content='Popover content goes here.'>Click Me</button>

Example 3 - In a table

You do not want to popover a <td> tag. It breaks the layout of the table when the popover displays. You want to wrap the item in a span etc.

Click the B column rows
Foo Bar
A B
C D 

//
// init on doc load
//
$(".popSpan").each(function(){
  $(this).popover({
    placement:'right',
    title:'popover in a table',
    content:'table poping'
  });
});

//
// markup
//
<table class="table table-condensed" style="width: 50%; margin: auto;">
  <thead>
    <tr>
      <th>Foo</th>
      <th>Bar</th>
    </tr>
  </thead>
  <tbody>
  <tr>
    <td>A</td>
    <td><span class="popSpan">B</span></td>
  </tr>
 <tr>
    <td>C</td>
    <td><span class="popSpan">D</span></td>
  </tr>  
</tbody>
</table>

Example 4 - Popover with HTML Content



Name:
Age: 
//
// css
//
<style>
  #popContent{
    display: none;
  }

  .popover {
    max-width: 600px;
  }
</style>

//
// init on doc load
//
$("#popEx4").popover({
  title:'HTML Content FTW!',
  content: popForm(),
  placement:'right',
  html: true
});

function popForm()
{
  var html = $("#popContent").html();
  return html;
}

//
// markup
//
<p><button id="popEx4" class="btn">Pop, pop-pop-pop</button></p>

<div id="popContent">
<table class="table table-striped">
  <tr>
    <td>Name:</td>
    <td><input type="text" size="10" /></td>
  </tr>
  <tr>
    <td>Age:</td>
    <td><input type="text" size="4" /></td>
  </tr>  
</table>
</div>
