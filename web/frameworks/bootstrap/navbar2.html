Navbar


Example 1 - Nav Bar Fixed To Top


Add class="navbar navbar-fixed-top" to the outter div and remember to account for the hidden area underneath it by adding at least 40px padding to the <body>. Be sure to add this after the core Bootstrap CSS and before the optional responsive CSS.

Create a full-width navbar that scrolls away with the page by adding class="navbar navbar-static-top". Unlike the .navbar-fixed-top class, you do not need to change any padding on the body.

Title
  • Home
  • Navbar 1
  • Profile
  • Settings

//
// style
//
<style>

  body
  {
    padding-top: 40px; 
  }

</style>

//
// markup
//
<div class="navbar navbar-fixed-top">
  <div class="navbar-inner">
    <a class="brand" href="#">Title</a>
    <ul class="nav">
      <li class="active"><a href="#">Home</a></li>
      <li ><a href="navbar.html">Navbar 1</a></li>
      <li><a href="forms.html">Profile</a></li>
      <li class="divider-vertical"></li>
      <li><a href="index.html">Settings</a></li>
    </ul>
  </div>
</div>
