Bootstrap Framework Demo

Chosen Select for Bootstrap

website

Note: The version of the plugin on the website doesn't work with jQuery 1.9+ due to $.browser() being deprecated. It fails on detecting IE6 & IE7. Use navigator.userAgent() for a workaround.

Example 1 - Regular Usage



// css
<link href="css/chosen.css" rel="stylesheet" media="screen">

// script
<script src="js/chosen.jquery.js"></script>

$(function() {
  $('#demo2').chosen();
});

Example 2 - Adjust Width



// css
<style type="text/css">
/*
 * select width
 */
.select-size75
{
    width: 75px;
}
</style>

// markup
<select id="demo4" class="select-size75">

// script
<script src="js/chosen.jquery.js"></script>

$(function() {
  $('#demo4').chosen();
});
