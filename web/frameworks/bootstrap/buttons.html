Buttons

Button styles can be applied to anything with class="btn"However, typically you'll want to apply these to only <a> and <button> elements

Example 1 - Default Buttons



<button class="btn">Default</button><br/>
<button class="btn btn-info">Info</button><br/>
<button class="btn btn-success">Success</button><br/>
<button class="btn btn-warning">Warning</button><br/>
<button class="btn btn-danger">Danger</button><br/>
<button class="btn btn-inverse">inverse</button><br/>

Example 2 - Default Sizes



<button class="btn btn-success btn-large">Large</button>
<button class="btn btn-success ">Default</button>
<button class="btn btn-success btn-small">Small</button>
<button class="btn btn-success btn-mini">Mini</button>

Example 3 - Disabled State with class="disabled"

A - disabled 

<button class="btn btn-warning">Active</button>
<button class="btn btn-warning disabled">Button - disabled</button>
<a class="btn btn-warning disabled">A - disabled</a>
