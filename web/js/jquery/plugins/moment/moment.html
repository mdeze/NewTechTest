Moment

A javascript date library for parsing, validating, manipulating, and formatting dates.

Web Site: http://momentjs.com/

The library does EVERYTHING you would want!

  • day of week from date
  • date math
  • convenient date math - last wednesday
  • many display options
  • calculate durations

Example 1

<script src="js/jquery.js"></script>

var n = moment();           // now
var d = moment(new Date()); // Date() in a moment wrapper - equiv to above

moment().format();                          // 2013-09-10T06:10:21-04:00 
moment().format('MMMM Do YYYY, h:mm:ss a'); // September 10th 2013, 6:10:21 am
moment().format('dddd');                    // Tuesday
moment().format("MMM Do YY");               // Sep 10th 13

Example 2 - parsing

// You can create a moment from a string that can be parsed by Date.parse.

var day = moment("Dec 25, 1995");

// the following ISO-8601 formats are supported across all browsers.
// e.g. "YYYY-MM-DD"

// If you know the format of an input string, you can use that to parse a moment.

var a = moment("12-25-1995", "MM-DD-YYYY");

// validation
var f = moment("2010 13", "YYYY MM").isValid(); // false (not a real month)
